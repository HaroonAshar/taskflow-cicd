About TaskFlow

TaskFlow is a lightweight task-management web app built to demonstrate a complete, modern CI/CD workflow — from feature branch to automated deployment.

⚡

Fast & Lightweight

Pure HTML, CSS and JavaScript — no frameworks, no build step. Loads instantly.

🎨

Modern UI

A responsive design system with light & dark themes, accessible components and smooth interactions.

💾

Works Offline

Tasks persist in your browser via localStorage — create, complete and delete without a backend.

🐳

Dockerized

Served from an nginx Alpine container for a tiny, production-ready image.

🔁

Continuous Integration

Every pull request builds and pushes a Docker image to Docker Hub via GitHub Actions.

🚀

Continuous Deployment

Merges into develop trigger an automatic deploy to Render through a webhook.

CI/CD Pipeline

  • 1 · Feature branch
    Work happens on feature/<name>-dev/<feature> branches.
  • 2 · Pull request → CI
    GitHub Actions builds the Docker image and pushes it to Docker Hub.
  • 3 · Merge to develop → CD
    A webhook triggers an automatic deployment to Render.
  • 4 · Live
    The updated site is served from the nginx container.

Tech Stack

HTML5 · CSS3 · Vanilla JavaScript · Docker · nginx · GitHub Actions · Render
