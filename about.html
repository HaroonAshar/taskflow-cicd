About TaskFlow

TaskFlow is a lightweight task management application that helps you organize your daily work efficiently.

Purpose

TaskFlow allows users to create tasks, track their progress, set priorities, and mark tasks as complete once finished.

How to Use

  • Go to Home to see an overview of all your tasks.
  • Go to Add Task to create a new task with a title, description, due date, and priority.
  • Visit Task List to view all tasks and filter them by status (All / Pending / Completed).
  • Click on a task in Task Detail to view its full information, mark it as complete, or delete it.
